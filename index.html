AVR Web Flasher

Flash an Intel HEX file to an ATmega328P over USB-serial using STK500 v1 — in the browser.

avrdude -c stk500v1 -p m328p -P <serial> -b 19200 -U flash:w:firmware.hex
Web Serial is not available in this browser.
This tool needs the Web Serial API (navigator.serial), which is only available in Chromium-based browsers — Chrome, Edge, Brave, or Opera — served over https:// or localhost. Please switch browsers and reload.
